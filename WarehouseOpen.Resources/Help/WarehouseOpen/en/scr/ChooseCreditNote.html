Void Credit Note

The Void issued invoice window (keyboard shortcut Alt+F7) contains a list of all invoiced sales documents. To void the invoice, select the document and click the OK button. Click the Cancel button to exit without selecting a document. 

After you void an invoice, you can issue a new one with the same number from the Credit Note window.
