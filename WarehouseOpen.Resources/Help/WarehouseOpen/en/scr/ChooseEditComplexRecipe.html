Recipes

Use the Recipes window to select a recipe, to add new recipes and to edit the recipes list. To select a recipe from the list, click the corresponding row in the table. Click the OK button to confirm. Click Cancel to exit without making the selection.


You can use the New, Edit, Delete buttons to add a new recipe and to edit or delete an existing one. To edit or delete an existing recipe you must select it and then confirm the action.
