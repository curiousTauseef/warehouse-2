Debit cash order

Use the Debit Cash Order window to edit the issued document and to enter additional data: date and number of the cash receipt note, the name of the person accepting the document, the name of the person giving the document, the name of the chief accountant, description and amount of the income. The cash receipt note certifies the incomes in the Cash Book.
