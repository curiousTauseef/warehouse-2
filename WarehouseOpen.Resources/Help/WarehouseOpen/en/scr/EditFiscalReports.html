Fiscal Reports

 

The window Fiscal reports is used to select the print type of the fiscal report. According to the reports that maintains fiscal device with which you work. It is necessary to select the type of report, and then click Print to be printed on the device. It is necessary to select the type of the report and after that button Print in order to be printed by the device.
