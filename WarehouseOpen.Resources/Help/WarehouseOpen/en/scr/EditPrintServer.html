Print Server

 

The window Print Server is created to allow access for printing fiscal receipts from other terminals to your fiscal device. It is necessary to work with one the same database in all workstations and in the other workstation to be selected device type �Server for printing� from menu Edit->Devices. This function makes easier the printing of fiscal receipts when there is more than 1 terminals and just one device.
